## USAGE
`modify [-r] [-l|-u] <dir/file names...>`  
`modify [-r] <sed pattern> <dir/file names...>`  
`modify [-h]`  
## ARGUMENTS
-r      run script with recursion    
-l      lowercase dir/file name(s)    
-u      uppercase dir/file name(s)    
-h      display this help manual
